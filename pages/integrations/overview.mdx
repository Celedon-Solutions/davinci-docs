## Overview

Integrations connect Davinci to external tools and services. Some integrations are configured from the Davinci Browser or account settings, while others are used inside the Davinci Editor after they are connected.

Use this section to understand what an integration connects to, where it is configured, and how it appears in project workflows.

## Where Integrations Are Managed

Integration settings can appear in different places depending on scope:

- **Account connections** — user-owned connections such as personal tokens and third-party account links.
- **Organization or team administration** — shared integrations or governance settings where available.
- **Editor project settings** — project-scoped integration and tool configuration, such as MCP servers or custom toolboxes.

If an integration is not visible, your deployment, role, or project settings may not allow access to that integration.

## Where Integrations Are Used

Most integration output is used inside the Davinci Editor. For example, imported CAD can become geometry objects, configuration attributes, references, or model context for the Davinci Agent and Model Review.
