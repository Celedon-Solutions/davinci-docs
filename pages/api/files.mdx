---
title: Files
description: List and download files attached to projects
---

## List Files

```http
GET /api/v2/programmatic/projects/:projectId/files?branch=main
```

Required scope: `projects:read`

Returns project reference objects that represent attached files.

## Download File

```http
GET /api/v2/programmatic/projects/:projectId/files/:fileId?branch=main
```

Returns raw file bytes. Response headers may include `Content-Type`, `Content-Disposition`, and `Content-Length`.

<CodeGroup>

```typescript TypeScript
const files = await client.projects.listFiles(projectId);
const download = await client.projects.downloadFile(projectId, files[0].id);
console.log(download.contentType, download.buffer.length);
```

```python Python
files = client.projects.list_files(project_id)
download = client.projects.download_file(project_id, files[0].id)
print(download.content_type, len(download.content))
```
