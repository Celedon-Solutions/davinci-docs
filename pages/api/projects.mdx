title: Projects
description: List accessible projects and fetch project metadata
---

## List Projects

```http
GET /api/v2/programmatic/projects?page=1&limit=50
```

Required scope: `projects:read`

Returns a flat paginated list of projects visible to the token owner.

```bash
curl -X GET "https://davinci-app.com/api/v2/programmatic/projects?limit=10" \
  -H "Authorization: Bearer dav_ak_live_your_token"
```

## Get Project

```http
GET /api/v2/programmatic/projects/:projectId
```

Returns project metadata, access reason, and effective permissions. A missing or unauthorized project returns `404 PROJECT_NOT_FOUND` so callers cannot probe private project existence.

<CodeGroup>

```typescript TypeScript
const project = await client.projects.get('project-id');
console.log(project.name, project.permissions);
```

```python Python
project = client.projects.get('project-id')
print(project.name, project.permissions)
```

</CodeGroup>
