## Overview

The Davinci Programmatic API currently exposes read-only access to projects. See the [generated API reference](/pages/api/reference) for the exact HTTP contract. It is intended for automation, reporting, indexing, and AI tools that need to inspect project structure and attached files.

The public v2 API does not expose sessions, agent messaging, git writes, project creation, or file uploads yet. Those surfaces should not be treated as available until they are documented here as v2 endpoints.

## Authentication

Use a personal access token in the `Authorization` header.

```bash
Authorization: Bearer dav_ak_live_your_token
```

All project reads require the `projects:read` scope.

## Quick Start

<CodeGroup>

```typescript TypeScript
import { DavinciClient } from '@celedon/davinci-sdk';

const client = new DavinciClient({
  apiKey: process.env.DAVINCI_API_KEY!,
  baseUrl: 'https://davinci-app.com',
});

const { projects } = await client.projects.list({ limit: 10 });
const tree = await client.projects.getTree(projects[0].id!, { branch: 'main' });
console.log(tree.tree.length);
```

```python Python
from davinci_sdk import DavinciClient

with DavinciClient(api_key='dav_ak_live_...', base_url='https://davinci-app.com') as client:
    result = client.projects.list(limit=10)
    tree = client.projects.get_tree(result.projects[0].id, branch='main')
    print(len(tree.tree))
```

</CodeGroup>
