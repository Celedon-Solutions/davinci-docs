## Get Project Tree

```http
GET /api/v2/programmatic/projects/:projectId/tree?branch=main
```

Required scope: `projects:read`

Returns a flat list of project objects. The first call may take longer because Routing Server auto-acquires a Design Engine instance.

## Get Object

```http
GET /api/v2/programmatic/projects/:projectId/objects/:objectId?branch=main
```

Returns the full payload for a project object. Object fields are type-specific; common fields include `id`, `type`, `name`, `parent`, `children`, and `documentation`.

<CodeGroup>

```typescript TypeScript
const tree = await client.projects.getTree(projectId, { branch: 'main' });
const object = await client.projects.getObject(projectId, tree.tree[0].id);
```
