## Description

Multiple users can work on the same Project together at the same time using Collaboration. Updates are rendered in near-real-time for each user, allowing for highly efficient teamwork within the Model.

Collaborators can be managed from two places:
- **Project Page (Davinci Browser)** — navigate to the project's page and select the **Settings** tab. The Collaborators section is listed there alongside project-level settings like visibility and deletion.
- **Editor (Project Settings)** — while inside the Editor, go to Project Settings > Collaboration.

## Managing Collaborators

Click the `Add Collaborator` button to invite a new collaborator to the project by email.

To remove a collaborator, find them in the collaborator list and use the actions menu (the three-dot icon) to remove them.

Pending invitations are listed separately and can be revoked before the invited user accepts.

## Joining a Project

After inviting a new Collaborator, they will receive an email inviting them to the project. Once they accept the invitation via the link in the email, they will be added to the project. 

If the invited user does not already have a Davinci account, they will need to create one before joining the Project. Some users may need to click the invitation link a second time after creating a new account to join the Project.

<Note>The invitation link is only valid for the account that the Project invite email was sent to. Check that you are signed in with the correct account if the invitation is not working properly.</Note>

## Roles

Davinci currently supports the following roles:
| Role | Description |
|------|-------------|
| `Owner` | The Owner is the user who created the Project. The Owner has all permissions and cannot be removed from the Project. |
| `Member` | Members can interact with the model, create, edit, and remove elements and objects, and chat with Davinci. Members can also invite other Davinci users to the Project. |
| `Pending` | A Pending user has been invited but has not yet accepted the invitation. Invitations can be revoked before they accept. |

<Note>The Role system will be expanded in future releases.</Note>

<Warning>Compute costs for tasks are charged to the user initiating that task. For team-level costing, a team account is required.</Warning>
