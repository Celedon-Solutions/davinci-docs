## Description
Documents are a containing object that holds text sections, tables, and figures that are assembled to build a traditional document structure.

<Note>The ordering of the text sections in the index define the section ordering. Sections can be nested to build subsections.</Note>

<Info>This is edited from the document view only</Info>

## Document View
The Document View provides a what-you-see-is-what-you-get (WYSIWYG) editor for composing the document, allowing for text formatting and structure management.
