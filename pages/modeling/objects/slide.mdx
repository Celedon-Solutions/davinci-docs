## Description
Slide objects are artifacts used to create presentations within the model. They allow engineers to communicate system design, analysis results, and project status to stakeholders in a familiar presentation format, directly linked to the live model data.

## View Types

| View | Description |
| :--- | :--- |
| [Document](#document-view) | Edit slide content and layout. |
| [Properties](/pages/user-guide/views/properties) | Edit the object's properties, attributes, and metadata. |
| [Relationships](/pages/modeling/common/relationships) | Explore the network of connections to other objects. |

## Document View
The Document View allows for the editing and layout of the slide content, including text, images, and live model data.
