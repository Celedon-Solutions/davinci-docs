## Overview

The Davinci Editor is the in-project workspace for engineering work. It opens when you select **Open Project** from a project page in the Davinci Browser.

Use the Editor to model systems, browse and edit objects, manage files and references, collaborate with teammates, run the Davinci Agent, perform model reviews, configure integrations, and export project data.

## Main Areas

- **Navigation bar** — switch between Home, Model, and Project areas.
- **Header bar** — search the model, navigate open views, invite collaborators, and access your account menu.
- **Index** — browse the model tree and Library.
- **Workspace** — open model objects, project settings, and generated views in tabs.
- **Right panel** — use the Davinci Agent, version control, search, comments, and validation tools.
