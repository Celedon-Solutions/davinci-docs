## Version 4.2.18 - Invitations, Org Projects & Scratchpad Fixes

This release standardizes invitations, improves organization-owned project visibility, and fixes several scratchpad interaction issues.

### Invitations & Notifications

- Standardized invitation routes and acceptance handling
- Added an invitations section on the notifications page
- Updated email join links to open the invitation review flow
- Added separate auto-accept settings for project and organization invites
- Fixed an invite failure path that could leave notification state inconsistent

### Organizations & Projects

- Added org policy controls for when users can create organizations
- Showed organization projects on the home page
- Listed inherited and direct project members together
- Added collaborator rows showing when and by whom a member was added
- Added an organization admin project section scoped to billing management permissions

### Scratchpad & Editor Fixes

- Added cycle detection for object moves
- Fixed selection crashes, relationship-layer crashes, and connection click behavior
- Improved scratchpad zoom and relationship selection performance
- Fixed chat formatting for agent questions
