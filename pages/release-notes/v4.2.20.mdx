## Version 4.2.20 - Permissions, Sessions & Billing Resilience

This release improves project permissions, session management, billing resilience, and modeling reliability before the 4.3 feature release.

### Permissions & Project Management

- Refactored effective project permissions resolution
- Improved member management UI
- Improved project administration workflows
- Fixed a Git merge path that could affect project data
- Reduced unnecessary permission fetches

### Sessions & Authentication

- Added user session management pages
- Added temporary password state handling
- Improved auth and notification templates

### Billing & Usage

- Added billing delinquency management
- Improved billing usage event resilience
- Fixed default tier behavior
- Updated usage and system health dashboards

### Modeling Fixes

- Fixed percentage rollup math and automatic percentage unit handling
- Improved branch deletion and stopping behavior
