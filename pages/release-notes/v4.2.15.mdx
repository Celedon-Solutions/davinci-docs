## Version 4.2.15 - Agent Planning Preview

This release introduces the first major pieces of agent planning, adds question handling in chat, and improves project export safety.

### [Agent Planning](/pages/editor/agent/planning)

- Added persisted agent plan state to projects
- Added plan display, restore, make, modify, and update behavior
- Added planning skill routing for systems engineering, software, mission design, proposal design, knowledge management, and source discovery
- Added an agent plan view with task cards and richer chat controls

### Agent Questions

- Added a question tool so the agent can ask for targeted clarification during a task
- Added UI handling for question cards in chat
- Improved chat input behavior for task cards

### Project Export & Safety

- Added project export sanitization
- Improved plan graph positioning and anchor links
- Fixed multi-factor authentication and storage configuration issues

### UI & Maintenance

- Improved right-panel and chat styling
