## Version 4.2.11 - Organization Administration

This release improves organization, storage, account, and authentication administration.

### Organizations, Roles & Storage

- Improved organization transfer and team-linking workflows
- Added org link request notifications
- Improved organization subscription and storage handling
- Added safeguards around canceled subscriptions and grace periods
- Expanded role assignment behavior while preserving last-admin protections

### Account & Authentication

- Added avatar image upload and user bubble image support
- Fixed SSO save behavior and removed mandatory Keycloak assumptions
- Updated breadcrumbs, project pages, and history view rendering
