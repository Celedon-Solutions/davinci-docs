## Version 4.1.3 - Settings Rebuild & Sequence View Improvements

This release includes a rebuilt project settings interface, improved sequence diagrams, account management enhancements, and refinements to the system review feature.

### Project Settings

- Project settings interface completely rebuilt with a new tabbed layout for easier navigation and future extensibility

### [Sequence](/pages/user-guide/views/sequence) View

- Significantly improved sequence diagram rendering with better layout, spacing, and visual clarity
- Updated code view rendering with better change detection

### Account Management

- Expanded account page with additional profile and settings sections
- User display name now updates reactively across comments and collaborator bubbles when changed
- Improved credit usage display

### System Review Refinements

- Refined review criteria for actions, attributes, interfaces, and linkages
- Better scoring for organizational structure reviews
- Improved code change detection during agent code updates

### Bug Fixes

- Fixed a recursive call stack issue
- Fixed persona deletion and persona selector behavior
