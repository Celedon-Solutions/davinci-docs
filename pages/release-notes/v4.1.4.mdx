## Version 4.1.4 - Smarter Agent Reading & Matrix Expansion

This release improves how the agent reads large artifacts, expands the allocation matrix, adds an undo-all button for agent changes, and refines state diagrams.

### Smarter Agent Reading

The agent can now read specific sections of large artifacts instead of loading them entirely, making it faster and more efficient when working with large documents, code objects, and spreadsheets.

### Allocation [Matrix](/pages/modeling/objects/matrix) View

- Matrix view now supports additional object types for cross-referencing
- Improved cell rendering with better visual feedback

### Chat Improvements

- **Undo All Button**: New button to undo all agent changes at once
- Fixed snippet anchor rendering for accurate object references in chat
- Improved UUID and object reference detection in messages
- Better user mention handling
- Updated session summary panel styling

### [State](/pages/user-guide/views/state) Diagrams

- Start state indicator nodes now render in state diagrams
- Improved layout with better initial state positioning

### Agent Improvements

- Agent now supports failure transition types when building state machines
- Fixed transition validation and formatting issues
- Improved persona expansion and editing in settings

### UI Improvements

- Long names in the name element are now capped for readability
- Improved text cell insert handling in tables
- Documents no longer block global keyboard shortcuts
- Various small fixes in project settings

### New & Updated Agent Tools

ReadDocumentBlocks, ReadObjectCode, ReadTableCells
