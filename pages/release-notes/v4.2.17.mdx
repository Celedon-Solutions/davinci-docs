## Version 4.2.17 - Task & Reference Handling Fixes

This release improves task rendering, scratchpad grouping, and reference reading.

### Tasks & Gantt

- Improved task dependency analysis and duration documentation
- Updated Gantt rendering, timeline export, and task display formatting
- Fixed action and state property rendering around task-related fields

### Scratchpad

- Improved grouping behavior
- Fixed arrow and selection behavior

### Reference Reading

- Added text-only reference reading for files where image-heavy context is too large
- Added OCR metadata to extracted image references

### Modeling Runtime

- Updated action processing and state machine code generation
- Improved unit cleanup behavior
