## Version 4.3.0 - API, Model Review & Planning

This release expands Davinci's developer API, adds model review workflows, improves agent planning, and deepens SysML interoperability.

### Developer API & SDKs

- Added [public API](/pages/api/introduction) documentation and official SDK examples
- Moved programmatic API access toward personal access tokens
- Improved API rate-limit behavior and API activity visibility

### [Model Review](/pages/editor/agent/model-review)

- Added model review tools for building, running, updating, and persisting model reviews
- Added deterministic validation, grounding, coverage views, issue synthesis, and review scheduling behavior
- Improved review start and stop cycles and hid older review toolbox UI

### [Agent Planning](/pages/editor/agent/planning)

- Improved persisted plan restore behavior
- Fixed chat restore behavior for thinking and tool-use nodes
- Improved action rendering and agent task display

### SysML, References & Exports

- Added [SysML v2 JSON export](/pages/editor/exports) work
- Improved [SysML export](/pages/modeling/overview/sysml-export) behavior
- Improved [reference handling](/pages/editor/files-and-references) and PDF anchor export
- Fixed [risk](/pages/modeling/objects/risk) and [Model Review](/pages/editor/agent/model-review) state persistence issues

### Browser & Admin

- Improved project last-activity tracking
- Improved organization storage management
- Improved login state messaging, notifications styling, and project sorting
