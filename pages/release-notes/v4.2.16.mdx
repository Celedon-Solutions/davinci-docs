## Version 4.2.16 - Scratchpad & Project Start Improvements

This release adds the scratchpad feature, improves project creation from prompts, and strengthens Design Engine readiness.

### Scratchpad

- Added scratchpad canvas, selection, movement, context actions, relationship mode, and minimap support
- Added relationship editing and hierarchy management in the scratchpad
- Fixed scratchpad rename, drag, copy/paste, and selection issues

### Project Creation

- Added prompt-to-project-name generation through the Routing Server
- Added starter prompt examples and first-prompt handling in chat
- Improved default prompt names and onboarding prompt content

### Relationships & Modeling

- Restricted relationship types based on the object types involved
- Prevented self relationships
- Improved child rendering and relationship handling

### Reliability & Tooling

- Added Design Engine readiness checks and safer instance startup behavior
- Added lint and dependency automation workflows
- Fixed several reference and undefined-variable issues
