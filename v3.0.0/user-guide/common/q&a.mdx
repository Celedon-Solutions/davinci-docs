title: FAQ
---

<AccordionGroup>
<Accordion title="How to view account details">
    Click the user icon in the top-right corner of the window to open the account page. Here, you can view credit usage, account details, and available actions.
</Accordion>

<Accordion title="How to create a project">
    From the Home tab, click the green `New Project` button. You'll be prompted to enter a name, and once an ID is assigned, the project will be ready to load.
</Accordion>

<Accordion title="How to add a collaborator to a project">
    After selecting and loading a project from the `Home` tab, go to the `Project` tab and navigate to the Collaborators section. Click the green `Add Collaborator` button to invite users.
</Accordion>

<Accordion title="How to reference objects">
    You can use `@` to reference objects throughout the application. This works in inline document text, table cells, and even in chat to link objects within a task request.
</Accordion>

<Accordion title="How to remove files">
    Access your stored project files from the `Project` tab under the Files section. From there, you can permanently delete files from the database.
</Accordion>

<Accordion title="How to view and undo AI changes">
    AI-generated changes are displayed in the chat response under each sub-task. Click the `View` button to open the change log, where you can undo or redo modifications.
</Accordion>

<Accordion title="How do I prevent editing of objects by the AI (or other users)?">
    You can lock objects from the index or the properties view in the workspace. This prevents both AI and human users from making changes to the object.

    <Note>Locking an object will also lock all of its descendants. You can unlock an object at any time.</Note>  

</Accordion>

<Accordion title="What Python packages can I use?">
    In the Code view, click the `Modules` button in the top-left corner of the header to see the available packages.
</Accordion>

<Accordion title="How do I export data?">
    You can export data for individual objects in various formats using the download button located at the bottom-left of the tab window for that open object in the workspace. Additionally, in the `Project` tab, you can download the entire project in SysML or JSON from the Project Summary page.

    <Note>Export formats are still in development and will be expanded in future releases.</Note>
</Accordion>

</AccordionGroup>
