## Description
Several hotkeys are available in the interface, which will perform different functions based on the focus or region that they are used in.

| key             | Name        | Impact                                                                                    |
| --------------- | -----       | ----------------------------------------------------------------------------------------- |
| `CTRL + C`      | Copy        | The name of the object.                                                                          |
| `CTRL + V`      | Paste       | The name of the object.                                                                          |
| `CTRL + I`      | Itatlic     | The name of the object.                                                                          |
| `CTRL + B`      | Bold        | The name of the object.                                                                          |
| `CTRL + U`      | Underline   | The name of the object.                                                                          |
| `CTRL + Z`      | Undo        | The name of the object.                                                                          |
| `CTRL + Y`      | Redo        | The name of the object.                                                                          |
