## Overview
The profile tab shows the details of current computation credit allowance and license information. 

### License Key
The license key can be updated or revalidated here. Clicking the validate button will check with Celedon servers. Should an issue occur an error message will display.

### Credit Balance
Both the monthly subscription and any additionally purchased credits are shown here. For details on how credits work ![read here.](/sections/overview/account.mdx)
