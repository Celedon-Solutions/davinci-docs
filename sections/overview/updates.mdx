## Updating the Client
- If a client update is available, a green "Update Ready" button will appear in the header.
- Click the button to automatically install any updates.

## Server Updates
- Davinci updates automatically.
- Server updates are scheduled for *Mondays and Thursdays from 11:30 - 11:59 pm Eastern* and may result in temporary disconnection to projects.
- For custom versioning and update management: [Contact us](https://www.celedonsolutions.com/contact)
