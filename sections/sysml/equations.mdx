## Syntax
Standard mathmatical operations can be used in values of attributes to build roll up values or other simple math operations. The syntax for this follows the standard format:

| Operation | Synatx |
| -------- | ----- | 
| Multiplication   | A*B |
| Division | A/B |
| Addition | A + B |
| Minimum | min(A, B) |
| Maximum | max(A, B) |

## Unit Referencing
Values can be defined inline with other attribute references as `10 (kg)` where the unit is placed between the ().
