## Description
Text objects define the major sections of a document, containing paragraphs. Hyperlinked objects can be defined inline in the text content using the `@` command.

<Note>Inline references of attributes can select either their value, unit, or name to display. Right click the inline bubble to change between display types. For all other objects only the name can be shown.</Note>

<Note>The ordering of the text sections in the index define the section ordering. Sections can be nested to build subsections.</Note>

<Info>This is edited from the document view only</Info>
